# Sistema de Gerenciamento de Loja de Roupas

Sistema desenvolvido em Java, com foco em organização, controle de estoque e gerenciamento de vendas.

O objetivo do projeto é simular o funcionamento real de uma loja de roupas, aplicando os principais conceitos de Programação Orientada a Objetos (POO).

## Tecnologias:
- Java 21
- Orientação a Objetos (POO)
- Intellij IDEA IDE
- Git e GitHub

## Funcionalidades:
- CRUD de Produtos: cadastrar, editar, listar e remover produtos
- Cadastro de Clientes com informações básicas
- Registro de Vendas, associando produtos e clientes
- Controle de Estoque, com atualização automática após vendas
- Cálculo do valor total das compras
- Listagem de dados organizada no console

## Estrutura do projeto:

- `Produto.java`: representa os itens da loja  
- `Cliente.java`: armazena informações dos clientes  
- `Venda.java`: registra as vendas realizadas  
- `Principal.java`: gerencia o fluxo principal e as interações do sistema  
- (Pacotes adicionais conforme módulos: Clientes, Funcionários, Empresa, Fornecedores, Roupas, Vendas)

-| Tabela          | Descrição                               |
